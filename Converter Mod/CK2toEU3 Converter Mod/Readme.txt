This is a converter mod for CK2 to EU3. It adds missing items to EU3 to make conversion a better experience.

Instructions:
	1. Add everything in the 'Mod Files' directory to your mod folder.
	2. Use the files under the 'Converter Files' directory to replace those in your converter directory.
	3. After conversion, place your converted save in the 'save games' directory of the mod folder.
	4. When starting EU3, make sure to select the converter mod.
